# NVIM Erlang Configuration

## Formatter

To format our **.erl** files we will use erlfmt.

> [!caution] 
>
> Make sure you have installed 'rebar3' 

## Rebar3 

To install Rebar3 go to [rebar](https://rebar3.org/docs/getting-started/) and install the executable:

``` bash
chmod +x rebar
./rebar local install
```
